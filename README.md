# Polygon Python Automation
Utility scripts for **Polygon (MATIC)** network automation and data validation.

## 🛠️ Tools
### 1. Price Monitor (`prueba.py`)
Real-time MATIC/USDT price tracker using non-restricted API endpoints to bypass regional blocks.

### 2. Wallet Validator (`Validador.py`)
Security script to verify Polygon wallet addresses. Cleans input strings and validates 42-character hexadecimal format.

## 🚀 Setup
```bash
pip install requests
python prueba.py
python Validador.py <address>

```
